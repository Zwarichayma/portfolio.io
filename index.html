Description: Aerotemp is an IoT-based temperature monitoring system developed using an ESP32 microcontroller and the DHT11 temperature sensor. The system continuously collects temperature and humidity data from the DHT11 sensor, which is transmitted to a mobile application built with React Native. The mobile app displays real-time temperature readings, logs the data for future analysis, and allows users to monitor the environment from anywhere. Data is stored in Firebase, ensuring reliable cloud storage and real-time updates across all connected devices.

Goals: Enable real-time temperature monitoring using the ESP32 and DHT11 sensor, transmit the data to a React Native mobile application**, and store it in Firebase for historical data analysis. The goal of Aerotemp is to provide users with an intuitive mobile interface for remote temperature monitoring, and ensure the system is scalable and reliable for both personal and industrial use.
